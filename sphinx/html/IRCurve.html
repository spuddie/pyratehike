IRCurve¶

class pyratehike.IRCurve(name: str, settings: RHSettings)¶

An IRCurve represents an interest rate curve.

Its constructor uses a RHSettings object to decide on the characteristics. After that, the bootstrap method constructs the curve. The main getter methods get_zero_rates, get_discount_factors and get_insta_forwards allow to extract data.

examples¶

>>> settings = RHSettings(
...     today_date = "2023-07-06",
...     interpolation_data_type = "logdf",
...     interpolation_method = "linear",
...     currency = "EUR"
... )
>>> curve_ois = IRCurve("ESTR", settings)

parameters¶

namestr

a name for the curve

settingsRHSettings

the settings to use

returns¶

None

>>> list_ibor = ListOfInstruments(
...     settings_ibor, ibor_path, curve_ois
... )
>>> curve_ibor.bootstrap(list_ibor.instruments)
